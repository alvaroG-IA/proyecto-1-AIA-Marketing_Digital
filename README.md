# IA Aplicada al Marketing Digital - Proyecto 1

Este es el primer proyecto de la asignatura de Inteligencia Artificial Aplicada (AIA), enfocado en optimizar estrategias de Marketing Digital mediante el uso de modelos de aprendizaje automático y análisis de datos.

## 🚀 Descripción del Proyecto

El objetivo de este proyecto es transformar datos de usuarios digitales en decisiones de negocio. Se busca entender el comportamiento del usuario, predecir tendencias de compra y mejorar el Retorno de la Inversión (ROI).

### Objetivos principales:

* **Análisis Exploratorio de Datos (EDA):** Identificación de patrones en el tráfico web y comportamiento de compra.
* **Predicción de Conversión:** Desarrollo de modelos de clasificación para prever la probabilidad de compra.

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Python 3.x
* **Librerías principales:**
    * `Pandas` y `NumPy` para manipulación de datos.
    * `Scikit-learn` para modelado predictivo y clustering.
    * `Matplotlib` y `Seaborn` para visualización de datos.
    * `Jupyter Notebooks` como entorno de desarrollo.

## 📁 Estructura del Repositorio

* `notebooks/`: Jupyter Notebooks con el análisis paso a paso (EDA, Preprocesamiento, Experimentos).
* `src/`: Donde se encuentra el main del proyecto y sus funciones auxiliares.
